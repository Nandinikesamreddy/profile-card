# Personal Profile Card

A simple, responsive HTML profile card showcasing personal information with modern styling.

## Description

This is a clean, minimalist profile card built with HTML and CSS. It features:
- Centered card layout with subtle hover effects
- Professional styling with shadow effects
- Responsive design that works on different screen sizes
- Personal profile information display

## Live here

[Click Here](https://nandhinikesamreddy.github.io/profile-card/)

## Features

- **Modern Design**: Clean white card with subtle shadows
- **Hover Effects**: Card lifts slightly on hover with enhanced shadow
- **Responsive Layout**: Adapts to different viewport sizes
- **Profile Section**: Displays name and profile image
- **Personal Information**: Brief bio/about section

## File Structure

```
Cards/
├── card.html
└── README.md
```

## How to Use

1. Open `card.html` in any modern web browser
2. The card will be centered on the page
3. Hover over the card to see the interactive lift effect

## Technical Details

### Technologies Used
- HTML5
- CSS3

### Key CSS Features
- Flexbox for layout
- CSS transitions for smooth animations
- Responsive units (viewport percentages)
- Box-shadow for depth effects
- Transform properties for positioning and hover effects

### Browser Support
Works in all modern browsers that support:
- CSS3 transitions
- Flexbox
- Modern CSS properties

## Customization

You can easily customize:
- Card dimensions (width, padding)
- Colors (background, text, borders)
- Shadow intensity
- Hover effects
- Profile image
- Personal information
